\doxysubsection{Detailed Description}
Provides Monte Carlo Pi calculation and visualization functions. 

This class implements the Monte Carlo method for calculating Pi using both sequential and parallel approaches. The parallel implementation uses Open\+MP to distribute the work across multiple threads.

The Monte Carlo method works by\+:
\begin{DoxyEnumerate}
\item Generating random points in a 2x2 square centered at the origin
\item Counting how many points fall inside a unit circle (radius 1)
\item Calculating Pi as 4 \texorpdfstring{$\ast$}{*} (points inside circle) / (total points)
\end{DoxyEnumerate}

The accuracy of the result improves with more points, but so does the computation time. The parallel implementation can significantly reduce computation time on multi-\/core systems.
