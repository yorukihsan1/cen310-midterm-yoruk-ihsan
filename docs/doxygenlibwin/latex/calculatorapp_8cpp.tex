\doxysubsection{Detailed Description}
Monte Carlo Pi calculation and visualization application. 

This application demonstrates the use of the Monte Carlo method for calculating Pi. It includes both sequential and parallel implementations, and provides performance analysis and visualization of the results.

The application performs the following tasks\+:
\begin{DoxyEnumerate}
\item Sequential Pi calculation with different point counts
\item Parallel Pi calculation with different thread counts
\item Convergence analysis of Pi approximation
\item Performance analysis and speedup calculation
\item Visualization of results through CSV files 
\end{DoxyEnumerate}
